## Estrutura do projeto 

#### Arquitetura:
 - A estrutura do projeto se dará pela arquitetura MVC, Model, view e controller. Contendo também outros diretórios e arquivos:
    - Public: Diretório com arquivos estáticos, como imagens, scripts, etc.
    - Config: Diretório com as configurações de acesso e conexão com os bancos de dados.
    - Model: Diretório contendo funções que podem ser reutilizadas em diferentes partes do código, com o objetivo de deixa-lo mais legível e facilitar a manutenção.
    - Views: Diretório com  as visualizações, o HTML, css e scripts javascript que renderizarão as visualizações dos conteúdos para os usuários.
    - Controllers: Diretório com o fluxo das diferentes funções que existiram dentro do projeto.
    - Routes: Diretório contendo as rotas necessárias para renderizar as viwes e chamar suas respectivas funções.
    - Middleware: (entender melhor).
    - app.js: Arquivo js que vai inicializar o aplicativo express
    - package.json: Arquivo de configuração do Node.js que inclui informações sobre o projeto e suas dependências.
